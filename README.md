# HUFS Uploader (부제 : 학교 홈페이지, 이렇게 구려도 괜찮은가?)

## Overview

학교 홈페이지 게시판의 업로드 기능이 너무 허접해서 따로 제작한 업로더입니다. 제가 만든 프로그램도 허접하지만 그래도 굳이 홈페이지에 들어가지 않고서도 여러 파일을 올릴 수 있다는 장점은 있습니다.

## Libraries

* electron, electron-builder
* react(create react app)
* foreman
* react-file-drop
